ndarray—Numpy的数组对象

  • 比python内建列表处理数据更快且使用内存更少。
  • ndarray是一个多维、同类型容器。
  • ndarray包含shape（形状）、ndim（维数）、dtype（元素数据类型）等属性。

1

import numpy as np

Numpy数组创建

  • np.array(object, dtype=None, copy=True, order=‘K’, subok=False, ndmin=0)
  • np.asarray(a, dtype=None, order=None) a:array like
  • np.ones(shape, dtype=None, order=‘C’) np.ones_like()
  • np.zeros(shape, dtype=float, order=‘C’) np.zeros_like()
  • np.empty(shape, dtype=float, order=‘C’) np.empty_like()
  • np.full(shape, fill_value, dtype=None, order=‘C’) np.full_like()
  • np.eye(N, M=None, k=0, dtype=, order=‘C’) 生成对角矩阵 N是行数 M是列数默认=N
  • np.identity(n, dtype=None) 生成单位阵

数组算术

  • 任何两个等尺寸的数组之间的算术操作都应用了逐元素操作的方式(包含加减乘除等)。
  • 带有标量的算术操作将计算参数传递到每个元素。
  • 同尺寸数组之间的比较会产生一个布尔值数组。

基础索引与切片

  • 一维数组基本索引与列表类似
  • 切片索引赋值：
    • 对列表以及数组索引赋值都直接更改原数组。
    • 列表切片索引只能赋值可迭代类型不能赋值标量，且可超出长度，相当于索引部分用赋的值代替。
    • 数组切片索引可赋值标量和可迭代对象，但可迭代对象只能是索引长度。

通用函数

称为ufunc 在数组中逐元素操作的函数。
一元通用函数：

  • np.abs()、fabs() 逐元素计算整数、浮点数、复数的绝对值
  • np.sqrt() 计算平方根 等价于arr ** 0.5
  • np.square() 计算平方 等价于arr ** 2
  • np.exp() 计算自然指数值 即 e的多少次方
  • np.log() 自然对数 。 log10()、log2()、log1p() 10, 2为底和 log(1+x)
  • np.sign() 计算符号值 1（整数） 0 -1
  • np.ceil()、floor() 向上取整和向下取整
  • np.rint() 保留整数位 并保持dtype
  • np.modf() 分别将数组的小数部分和整数部分按数组形式返回
  • np.isnan 判断元素是否为NaN 返回一个布尔数组
  • np.logical_not() 按位取反 等价于 ~arr
  • cos cosh sin sinh tan tanh 常规双曲三角函数
  • arc… 反三角

二元通用函数：

  • add、multiply、divide、floor_divide 加、乘、除、整除
  • substract 在第二个数组中将第一个数组包含的元素去除
  • power 将第二个数组元素作为第一个数组对应元素的幂次方
  • maximum、fmax 逐元素计算最大值,fmax忽略NaN
  • minimum、fmin 。。。
  • mod 求模,即除法余数

唯一值与其他逻辑集合

  • np.unique(x) x中的唯一值并排序
  • np.intersect1d(x,y) 计算x，y的交集并排序
  • np.union1d(x,y) 计算x，y的并集并排序
  • np.in1d(x,y) 计算x中的元素是否包含在y中，返回布尔值数组
  • np.setdiff1d(x,y) 差集， 在x但不在y中的x元素
  • np.setxor1d(x,y) 异或集， 在x或y中但不同时在x，y中

线性代数

常用函数：

  • np.diag(x) 将一个方阵的对角（或非对角）元素作为一维数组返回,
    或将一维数组转换为一个方阵，非对角线上有零点
  • x.dot(y), np.dot(x,y) 矩阵点乘
  • np.trace(x) 计算对角元素和
  • np.linalg.det(x) 计算矩阵的行列式
  • np.linalg.eig(x) 计算方阵的特征值和特征向量
  • np.linalg.inv(x) 计算方阵的逆矩阵
  • np.linalg.pinv(x) 计算矩阵的Moore-Penrose伪逆
  • np.linalg.qr(x) 计算QR分解
  • np.linalg.svd(x) 计算奇异值分解（SVD）
  • np.linalg.solve(A, b) 求解x的线性系统Ax = b，其中A是方阵
  • np.linalg.lstsq(a, b, rcond='warn')计算Ax = b的最小二乘解

伪随机数生成

np.random：

  • seed() 向随机数生成器传递随机状态种子
  • permutation 返回一个序列的随机排列，或返回一个乱序的整数范围序列。
    x : int or array_like. integer:randomly permute np.arange(x).
    array:make a copy and shuffle the elementsrandomly.
  • shuffle 随机排列一个序列
  • rand 从均匀分布中抽取样本
  • uniform distribution over [0, 1) 参数为shape
  • randint 根据给定的由高到低的范围抽取随机整数。
    randint(low, high=None, size=None, dtype=‘l’) high=None时 [0,low)
  • randn 从0，1的正态分布中抽取样本 参数size
  • binomial 从二项分布中抽取样本 binomial(n, p, size=None)
  • normal 从正态分布中抽取样本 normal(loc=0.0, scale=1.0, size=None)
  • beta 从beta分布中抽取样本
  • chisquare 从卡方分布中抽取样本
  • gamma 从伽马分布中抽取样本
  • uniform 从均匀[0，1) 分布中抽取样本
